Dishwashing
The Dishwashing cover project was created in January 2025 as requested by my sister. This project was created by using PhotoShop and it is called compositing, by using different images to compose a single useful image. It was created to help my sister's husband to sell dishwashing liquid to the market for the organization of PWD in their barangay.

Cycling Jersey
The team Machinist jersey this is my project to help my co-cyclist to create a new jersey for our private group. They give me the preferred color of the jersey; that's why it's colored black and yellow-green. It's created using Adobe Illustrator for high-quality design. It took me almost 2 months to finish this design because I have 0 knowledge of Adobe Illustrator, so I needed to study first.

Water refilling Station logo design
I made this logo for my cousin, who runs his 1st water refilling business. This is my first paid design; this was created using Adobe PhotoShop, Illustrator, Blender, and some Microsoft Office. Sample exercises

Basic Calculator
This is a basic calculator that can perform MDAS, including modulo and square root operations. When I was creating this exercise, I gained more knowledge in JavaScript that the function eval is the easy way to compute the operation, but it's too risky because the user can input a malicious script on it.

Search algorithms
This exercise is created to search for a number in an array by using linear search and binary search. The linear search is an algorithm that finds data by index in succession, from index 0 to the max index. Binary search works by repeatedly dividing a problem into two smaller sub problems. This is used to refresh my knowledge in programming.

To Do List
The function of this website is to create a list of work, and you can modify it by marking it complete or by deleting it. The gain knowledge in this exercise is to understand the object called localStorage. This storage can save data in your local browser storage that can't be removed even if you refresh or close your website. I also learned about eventListener, which can catch your action on your website, and also the sessionStorage, which is almost like a session in PHP, where the difference is where it is stored.
